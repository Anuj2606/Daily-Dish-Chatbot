# 🍽️ Daily Dish AI Chatbot

An AI-powered restaurant customer support chatbot built using a Multi-Agent RAG (Retrieval-Augmented Generation) architecture.

The chatbot uses semantic search, FAISS vector database, and sentence-transformer embeddings to retrieve accurate answers from restaurant FAQ documents.

## Features

- Multi-Agent Workflow
- Semantic Search
- FAISS Vector Database
- Sentence Transformer Embeddings
- Conversational Memory
- Streamlit Chat Interface
- PDF-Based Knowledge Retrieval

## Architecture

1. Query Understanding Agent
2. Document Retrieval Agent
3. Memory Agent
4. Response Generation Agent
